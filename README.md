# Calculator_with_react

A simple calculator built with React to practice useState, component-based architecture, and event handling. Supports basic arithmetic operations with a clean UI.

## Features

- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
  - Percentage (%)
- Clear all input (C)
- Delete last character (⌫)
- Decimal number support
- Error handling for invalid expressions
- Clean and user-friendly interface

### Prerequisites

Make sure you have the following installed:

- Node.js
- npm (Node Package Manager)
